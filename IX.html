Problemática

Describa el problema a resolver:

La sala de té Layco necesita una aplicación web para gestionar eficientemente las reservaciones de eventos. Los usuarios deben poder ingresar datos personales, seleccionar el tipo de evento y la fecha, indicar la cantidad de personas y elegir un paquete. La aplicación debe asignar un empleado específico y un salón adecuado según la capacidad. Además, debe calcular el costo del evento, incluyendo IVA y el costo total, así como gestionar los requisitos de anticipo y depósito. Todo el proceso debe ser claro y accesible para el cliente, garantizando una administración efectiva de las reservaciones.
